我主要关注 Java 后端、自动化工具和实用型项目开发。相比单纯堆功能，我更在意把零散需求整理成稳定、可维护、能长期使用的小系统。

- 正在整理和维护老乞丐提醒系统相关项目
- 关注后端业务系统、微服务、RPC、数据处理和脚本自动化
- 也在练习 Flutter / Kotlin 移动端项目和轻量 Web 页面
